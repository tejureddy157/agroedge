# 🌾 About AgroEdge

**AgroEdge** is a modern AgriTech platform that blends technology with agriculture to create a seamless digital experience for farmers, agricultural businesses, and rural entrepreneurs.

The platform is designed to simplify agricultural operations, improve productivity, and showcase innovative farming solutions through a clean, responsive, and engaging user interface.

Inspired by the growing digital transformation in agriculture, AgroEdge focuses on making farming smarter, more accessible, and future-ready. Similar platforms emphasize practical training, business support, and modern farming techniques to improve sustainability and profitability.

# 🚀 Getting Started

### Clone the Repository

```bash
git clone https://github.com/tejureddy157/agroedge.git
```

### Navigate to the Project

```bash
cd agroedge
```

### Install Dependencies

```bash
npm install
```

### Start Development Server

```bash
npm run dev
```

### Production Build

```bash
npm run build
```
